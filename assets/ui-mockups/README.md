# UI Mockups

这些图片根据当前《中国古代官场模拟 × CK3 式结构 × 卡牌构筑玩法设计》和现有网页布局生成，可作为项目界面美术参考或临时展示素材。

## 文件

- `main-gameplay.png`：主对局界面，覆盖顶部状态、左侧人物、中央事务、右侧关系朝局、底部手牌。
- `relationship-court.png`：关系/朝局视图，展示人物网络、信任、猜忌、亏欠、把柄和清算风险。
- `deck-library.png`：卡组/牌库视图，展示牌库筛选、卡牌网格、卡牌详情、升级和异化操作。
- `ending-biography.png`：生平/结局视图，展示身后评定、仕途时间线和导出生平等操作。

## 使用建议

- 先作为视觉目标图使用，不建议直接依赖图片里的文字作为正式 UI 文案。
- 当前项目页面已经有相近的信息架构，后续可以把这些图里的元素拆成真实 HTML/CSS 组件。
- 若用于正式展示，可优先裁切出背景、卡牌框、卷轴、人物详情板等局部元素。
