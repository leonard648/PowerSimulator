## 1. 一句话概念

一款以中国古代官场为主题的角色驱动策略模拟游戏。玩家不是扮演抽象国家，也不是单纯做升官养成，而是扮演一名具体士人，在科举、任官、党争、地方治理、皇权压力、家族责任与历史危机之间，通过卡牌构筑经营自己的能力、关系、名声、权力与命运。

核心体验可以概括为：

> 用一生有限的时间，构筑一套属于自己的为官方法，并承担它带来的历史评价。

## 2. 设计支柱

### 2.1 人物而非国家

游戏的主角是一个人，而不是国家机器。玩家真正管理的是：

- 自身能力：才学、政务、权谋、口才、操守、体魄。
- 社会评价：清名、能名、文名、权名、恶名、贪名。
- 关系网络：君臣、师生、同年、门生、姻亲、上司、下属、政敌、地方士绅。
- 心理状态：压力、野心、恐惧、良心负担、倦怠。
- 人生时间：青春、壮年、中年、晚年，每一步都不可逆。

玩家不是在问“怎样赢下这局”，而是在问“我这个人最终会被怎样记住”。

### 2.2 官职而非等级

官职不是线性升级数值，而是一个个带有权限、约束和风险的位置。

知县拥有直接治理权，但被钱粮、士绅、考成、上司和民变牵制。御史拥有弹劾权，却极易树敌。翰林清贵近君，但缺少实权。户部掌钱粮，却容易卷入亏空与贪腐。内阁接近权力顶点，也最容易被清算。

每个官职都是一个“玩法包”：

- 解锁不同事件池。
- 改变可使用的卡牌类型。
- 改变手牌上限、行动点、政策槽或关系入口。
- 带来固定敌人、固定责任和固定考核。

### 2.3 关系驱动事件

世界不是一张等待征服的地图，而是一张不断变化的人际和制度网络。

同一件事，在不同关系状态下会变成不同事件：

- 皇帝信任高：直言进谏可能成为改革机会。
- 皇帝猜忌高：同样的进谏可能被视为结党。
- 士林评价高：弹劾权贵会获得清议支持。
- 士林评价低：弹劾会被认为是投机攀附。
- 地方士绅亲近：推行赋役改革更容易落地。
- 地方士绅敌视：改革可能引发粮价、讼案和谣言。

游戏的叙事由人物、制度、关系和卡组共同生成。

### 2.4 卡牌是政治资源

本作的卡牌不是魔法攻击，而是官场中的行动、话术、制度工具和人情资源。

一张卡可能是一篇奏疏、一封私信、一名幕僚、一条把柄、一次宴请、一项政令、一场廷辩、一笔赈银、一份案卷，甚至是一种为官风格。

卡组代表玩家“这个人解决问题的方式”。长期使用什么卡，就会塑造什么人生。

### 2.5 胜利不是唯一目标

游戏不设唯一胜利条件。结局以历史评价和人生总结呈现：

- 一代名臣。
- 清流孤臣。
- 能臣权臣。
- 酷吏干才。
- 贪墨巨蠹。
- 党争牺牲品。
- 身后平反者。
- 苟全善终者。
- 生前显赫、死后清算者。

玩家可以追求高位，也可以追求清名、改革成功、地方安定、家族延续、著书立说或身后名。

## 3. CK3 式结构映射

本作借鉴 CK3 的不是中世纪封建题材，而是其“人物、头衔、关系、事件、长期评价”共同运转的结构。

| CK3 结构 | 官场模拟转译 | 玩法作用 |
|---|---|---|
| 人物属性 | 才学、政务、权谋、口才、操守、体魄 | 决定卡牌效率、事件选项和风险抵抗 |
| 特质 | 刚直、圆滑、惜名、嗜财、多疑、宽仁、酷烈 | 改变卡牌费用、事件倾向和压力来源 |
| 头衔 | 官职、差遣、兼职、钦差身份 | 决定权限、事件池、收入、敌人和考核 |
| 领地 | 管辖区域、部院职权、项目责任 | 玩家不拥有土地，而是临时治理职责 |
| 宫廷 | 朝廷、内阁、六部、都察院、翰林院 | 中央政治场与信息网络 |
| 封臣关系 | 上司、下属、属官、门生、地方士绅 | 决定命令能否落地 |
| 好感度 | 亲近、信任、畏惧、亏欠、怨恨 | 触发支持、背刺、举荐、弹劾 |
| 牵制/把柄 | 案卷、私信、贪墨证据、旧日恩怨 | 可换取合作，也可能造成清算 |
| 压力 | 心病、良心负担、恐惧、倦怠 | 限制极端玩法，制造人物代价 |
| 继承 | 身后评价、家族命运、门生余波 | 主模式不换人，但保留历史回声 |

## 4. 世界层级

### 4.1 时代剧本

每个剧本定义一个历史压力场，而不是还原所有朝代细节。

剧本字段包括：

- 朝代阶段：开国、守成、改革、党争、边患、财政危机、末世。
- 皇帝性格：勤政、猜忌、宽纵、好名、怠政、求治、刚愎。
- 制度状态：财政健康、吏治清明、军备、灾害、民心、清议、朋党烈度。
- 主导议题：税制改革、边防、漕运、盐法、灾荒、宦官、外戚、党争、继储。
- 关键人物：皇帝、首辅、权臣、清流领袖、宦官、边将、地方大族。

### 4.2 朝廷结构

朝廷是游戏的核心政治地图。它由职位、人物和派系关系组成。

主要节点：

- 皇帝：最终任免权与合法性来源。
- 内阁或中枢：政策协调与最高斗争场。
- 六部：吏、户、礼、兵、刑、工，各自提供不同卡池和事件。
- 都察院：弹劾、巡按、监察、清议入口。
- 翰林院：文名、近君、诏令、储君教育入口。
- 地方督抚：财政、治安、边防、民变入口。
- 宦官或内廷：信息、密旨、宠信、黑箱权力入口。
- 士林舆论：不直接任免，却影响名声、弹劾和身后评价。

### 4.3 地方结构

地方不是传统 4X 领土，而是玩家任期内需要处理的责任现场。

每个地方有：

- 民生：粮价、灾害、治安、人口流动。
- 财政：税粮、亏空、徭役、盐课、漕运。
- 士绅：合作度、垄断度、书院、宗族、乡约。
- 官僚：佐贰、胥吏、捕快、幕友、仓官。
- 风险：民变、讼案、瘟疫、盗匪、兵灾、谣言。

地方治理结果会影响玩家进入中央后的声誉：能吏、酷吏、仁官、贪官、纸面政绩。

## 5. 核心循环

### 5.1 人生循环

```text
出身与志向
  -> 科举与入仕
  -> 初任官职
  -> 任期事件与卡组塑形
  -> 升迁、调任、贬谪或致仕
  -> 进入更高风险的位置
  -> 改革、党争、清算、病老
  -> 身后评价
```

玩家每一阶段都会获得新卡，也会把某些卡固化为人格和名声。

例如，长期使用“严刑催科”“株连胥吏”“密折告发”，玩家会更容易解决短期问题，但角色会逐渐获得“酷烈”“多疑”“权名上升”“士林敌视”等长期后果。

### 5.2 年度循环

每年分为四个阶段：

1. 春：任命、考课、科举、朝会。
2. 夏：政务推进、灾害、财政征收。
3. 秋：案件、弹劾、巡查、边防。
4. 冬：总结、举荐、清议、家族事务、健康检定。

每个季度流程：

```text
读取官职权限与时代压力
  -> 抽取公共事件、关系事件、官职事件
  -> 玩家选择处理优先级
  -> 抽取手牌
  -> 使用卡牌解决事件或推进长期项目
  -> 结算显性收益与隐性代价
  -> 更新人物关系、名声、压力和制度变量
```

### 5.3 单事件循环

事件不是一次点击选项，而是一个小型卡牌局。

```text
事件登场
  -> 显示目标、阻力、涉事人物和期限
  -> 玩家抽取手牌
  -> 打出行动卡、人物卡、资源卡或话术卡
  -> 对方势力反制
  -> 结算成功度、代价和后续钩子
```

事件局通常持续 1 到 3 回合。短事件适合叙事节奏，长事件适合改革、审案、廷议和党争。

## 6. 卡牌系统

### 6.1 卡牌定位

卡牌代表玩家可调用的政治动作与资源。它既是战术工具，也是角色成长记录。

卡牌基本字段：

| 字段 | 说明 |
|---|---|
| 名称 | 显示名称，如“上疏直谏”“宴请乡绅”“查封账册” |
| 类型 | 行动、人物、资源、政令、奏疏、把柄、心病、污点 |
| 领域 | 才学、政务、权谋、人情、法度、财政、清议、家族 |
| 费用 | 时间、精力、银两、名声、人情、压力、良心债 |
| 效果 | 推进目标、降低阻力、改变关系、获得证据、转移风险 |
| 标签 | 清流、酷吏、能吏、朋党、近君、地方、改革、贪腐等 |
| 后果 | 立即结算或埋入长期事件池 |

### 6.2 牌库区域

| 区域 | 含义 |
|---|---|
| 抽牌堆 | 当前为官风格与可调用资源 |
| 手牌 | 此刻能想到、能做、能动用的选择 |
| 弃牌堆 | 本轮已使用或暂时错过的资源 |
| 封存区 | 致仕、贬谪、失势或过时后无法使用的卡 |
| 污点区 | 贪墨、把柄、亏欠、旧案，会被敌人调用 |
| 人脉区 | 长期人物牌，不进入普通抽牌堆，按条件触发 |
| 政策区 | 长期项目牌，占用槽位并持续产生效果和风险 |

### 6.3 卡牌类型

| 类型 | 例子 | 主要用途 | 典型代价 |
|---|---|---|---|
| 奏疏牌 | 上疏直谏、密折陈情、代拟诏书 | 影响皇帝和朝局 | 皇帝猜忌、树敌 |
| 政务牌 | 清丈田亩、整顿驿传、修筑水利 | 推进治理项目 | 时间、财政、地方反弹 |
| 法度牌 | 查封账册、按律问罪、弹劾不法 | 打击对手或清理弊政 | 怨恨、报复、压力 |
| 人情牌 | 拜访座师、宴请乡绅、托付同年 | 获取支持或缓和阻力 | 清名下降、人情债 |
| 权谋牌 | 离间朋党、密访内侍、扣留案卷 | 操作信息与风险 | 污点、恐惧、恶名 |
| 清议牌 | 公开讲学、诗文传名、士林联署 | 提升士林评价 | 行动慢、容易激怒中枢 |
| 财政牌 | 开源节流、借拨库银、商税加派 | 解决钱粮压力 | 民怨、贪腐暴露 |
| 家族牌 | 族中资助、姻亲求情、子弟入学 | 补充资源或打开关系 | 家族牵连、裙带恶名 |
| 心病牌 | 夜不能寐、恐惧清算、良心不安 | 负面牌，污染手牌 | 需要休养或忏悔清除 |
| 污点牌 | 收受冰敬、隐匿亏空、纵容门生 | 短期强力，长期危险 | 被弹劾、清算、身后恶评 |

### 6.4 构筑来源

玩家通过人生选择获得卡牌：

- 科举路径：获得才学、文章、清议类卡。
- 地方治理：获得政务、财政、民生、胥吏类卡。
- 御史经历：获得法度、弹劾、证据、清流类卡。
- 部院任职：获得政策、流程、官样文章、协调类卡。
- 近君经历：获得密折、诏令、内廷、人主心理类卡。
- 党争经历：获得权谋、把柄、派系、人情债类卡。
- 贪腐妥协：获得强力资源牌，同时加入污点牌。
- 病老退让：获得著述、讲学、托孤、身后名类卡。

### 6.5 卡牌升级与异化

卡牌不是简单加数值，而是根据使用方式变形。

| 原始牌 | 升级方向 A | 升级方向 B | 异化方向 |
|---|---|---|---|
| 上疏直谏 | 慷慨陈词：清名更高 | 委婉陈情：风险更低 | 沽名邀直：士林加成、皇帝猜忌 |
| 查封账册 | 铁案如山：证据更强 | 留有余地：关系伤害低 | 罗织罪名：强力但增加恶名 |
| 宴请乡绅 | 宾主尽欢：缓和地方 | 利益交换：获得资源 | 朋比为奸：加入污点 |
| 清丈田亩 | 严格丈量：财政提升 | 渐进改革：民怨较低 | 苛政催逼：短期暴增、长期民变 |
| 密访内侍 | 获得风声：预警增强 | 暗通消息：解锁内廷 | 内外交结：清算风险上升 |

## 7. 事件与对抗设计

### 7.1 事件不是敌人血条

事件使用“阻力条”而非生命值。不同阻力代表不同问题来源。

常见阻力：

- 上意不明：皇帝态度未定。
- 士绅掣肘：地方利益集团阻挠。
- 钱粮不足：财政资源不够。
- 胥吏欺瞒：基层执行失真。
- 清议沸腾：舆论压力过高。
- 朋党反扑：政治敌人正在集结。
- 民怨积累：百姓承受能力下降。
- 证据不足：案件无法坐实。
- 期限逼近：再拖就会自动恶化。

玩家不是单纯把所有条清空，而是决定“优先压哪一边，放任哪一边变成后果”。

### 7.2 事件示例：县仓亏空案

背景：

玩家任知县。秋粮将至，县仓账册显示存粮充足，但实际开仓时发现大量亏空。上司要求限期补足，地方士绅暗示可以共同遮掩，胥吏推称前任遗留。

阻力：

| 阻力 | 初始值 | 说明 |
|---|---:|---|
| 钱粮缺口 | 8 | 不补足会影响考成 |
| 证据不足 | 6 | 无法确认责任人 |
| 士绅掣肘 | 5 | 本地大族牵涉其中 |
| 期限逼近 | 4 | 三回合后上司巡查 |

可用卡牌示例：

| 卡牌 | 效果 | 代价 |
|---|---|---|
| 查封账册 | 证据不足 -3 | 胥吏怨恨 +2 |
| 宴请乡绅 | 士绅掣肘 -2，获得银两 | 清名 -1，人情债 +1 |
| 严刑催供 | 证据不足 -4 | 民怨 +2，酷吏标签 +1 |
| 借拨库银 | 钱粮缺口 -4 | 财政隐患 +3 |
| 上报实情 | 期限重置，皇帝或上司信任检定 | 能名可能下降 |
| 私下补账 | 钱粮缺口 -5 | 污点区加入“隐匿亏空” |

结局分支：

- 依法追查：获得能名和清名，树立地方敌人。
- 遮掩过关：考成安全，污点进入未来事件池。
- 借银救急：短期稳定，财政隐患延后爆发。
- 严刑速断：快速破案，民怨与酷吏评价上升。
- 上报求援：风险转移到上级关系，可能升也可能贬。

## 8. 资源与数值

### 8.1 核心资源

| 资源 | 含义 | 过高或过低的风险 |
|---|---|---|
| 时间 | 人生与任期消耗 | 时间耗尽会进入病老、致仕或清算 |
| 精力 | 当前行动能力 | 过低会抽到心病牌 |
| 银两 | 私人资源与周转能力 | 来源不清会产生污点 |
| 清名 | 士林与后世评价 | 过高会降低妥协空间 |
| 能名 | 办事能力评价 | 过高会被推到危险岗位 |
| 权势 | 调动他人的能力 | 过高会引发皇帝猜忌 |
| 皇帝信任 | 中枢安全感 | 过低容易被罢免或下狱 |
| 士林评价 | 舆论与清议支持 | 过低会被视为奸佞 |
| 民心 | 地方稳定与仁政评价 | 过低触发民变或身后恶评 |
| 压力 | 心理代价 | 过高加入心病牌，可能病倒 |
| 污点 | 可被攻击的隐患 | 累积后触发弹劾、清算、抄家 |

### 8.2 名与实

本作重点区分“名”和“实”。

- 清名高，不代表能办成事。
- 权势高，不代表安全。
- 能名高，不代表道德清白。
- 皇帝信任高，不代表士林认可。
- 民心高，不代表考成漂亮。

很多强力卡都会在“名”和“实”之间做交换。

例如：

- “密折告发”提高皇帝信任和权势，但降低士林评价。
- “拒收陋规”提高清名，但降低银两和地方合作。
- “摊丁入亩”提高财政制度健康，但短期激怒士绅。
- “息事宁人”降低当前风险，但提高制度淤积。

## 9. 人物系统

### 9.1 基础属性

| 属性 | 作用 |
|---|---|
| 才学 | 科举、文章、经义、诏令、讲学 |
| 政务 | 治理、财政、工程、考成、流程 |
| 权谋 | 信息、把柄、离间、暗线、反制 |
| 口才 | 廷辩、说服、宴会、调停 |
| 操守 | 抵抗贪腐、维持清名、承受诱惑 |
| 体魄 | 抵抗压力、疾病、长期任事 |

### 9.2 性格特质

特质不只是加减数值，还改变压力来源。

| 特质 | 优势 | 压力来源 |
|---|---|---|
| 刚直 | 直谏、弹劾、清名收益高 | 妥协会增加压力 |
| 圆滑 | 人情牌费用低，关系修复强 | 公开冲突会增加压力 |
| 惜名 | 清议收益高，污点伤害大 | 使用脏牌压力翻倍 |
| 嗜财 | 财政和银两牌收益高 | 拒绝利益会增加压力 |
| 多疑 | 反制阴谋强，难被偷袭 | 信任他人会增加压力 |
| 宽仁 | 民心收益高，酷吏牌受限 | 严刑和牺牲会增加压力 |
| 酷烈 | 快速清理阻力 | 民怨和恶名更容易累积 |
| 恬退 | 致仕、著述、保全结局更好 | 追求高位会增加压力 |

### 9.3 关系维度

关系不是单一好感度，而由多个维度构成。

| 维度 | 含义 |
|---|---|
| 亲近 | 愿意帮忙 |
| 信任 | 相信你的承诺 |
| 畏惧 | 不敢反抗 |
| 亏欠 | 欠你人情或利益 |
| 怨恨 | 想要报复 |
| 把柄 | 你掌握对方风险 |
| 牵连 | 对方出事会拖累你 |

一个人可能既亲近你，又怨恨你；既畏惧你，又握有你的把柄。这会制造更真实的官场关系。

## 10. 官职系统

### 10.1 官职是权限包

当前 Demo 实装了前三个阶段：翰林、知县、御史。部院官、巡抚/总督、尚书、内阁/中枢仍是后续扩展方向。

| 官职阶段 | 权限 | 主要事件 | 主要风险 | 代表卡池 |
|---|---|---|---|---|
| 举人/进士 | 入仕资格、师门同年 | 科举、座师、馆选 | 名次、出身、派系预设 | 文章、拜谒、同年 |
| 翰林 | 近君、文名、诏令 | 修书、讲读、储君教育 | 无实权、卷入中枢 | 清议、诏令、讲学 |
| 知县 | 直接治理、钱粮刑名 | 税粮、讼案、灾害、治安 | 胥吏、士绅、亏空 | 政务、法度、地方人情 |
| 御史 | 弹劾、巡按、监察 | 参劾、巡查、风闻 | 树敌、被反噬 | 弹劾、证据、清名 |
| 部院官 | 流程、政策、档案 | 考成、预算、任免 | 背锅、拖延、派系 | 公文、协调、档案 |
| 巡抚/总督 | 地方大权、军政财政 | 民变、边患、改革 | 功高震主、地方盘根 | 军政、财政、强制 |
| 尚书 | 部门最高责任 | 全国政策、官员任免 | 派系围攻、制度责任 | 政策、任免、廷议 |
| 内阁/中枢 | 议政、票拟、最高协调 | 改革、储位、清算、党争 | 皇帝猜忌、身后清算 | 诏令、权谋、改革 |

当前官职参数：

| 官职 | 年段 | 官阶名 | 手牌上限 | 每季精力 | 升迁阈值 | 下一官职 | 当前任期目标 |
|---|---|---|---:|---:|---:|---|---|
| 翰林 | 1-4 年 | 翰林编修 | 5 | 4 | 28 | 知县 | 建立文名、近君关系与士林声望 |
| 知县 | 5-8 年 | 七品知县 | 5 | 5 | 64 | 御史 | 在钱粮、讼案、灾荒和士绅之间完成治理 |
| 御史 | 9-12 年 | 监察御史 | 6 | 5 | - | - | 用证据、弹劾与清议在朝局中立身 |

### 10.2 任期考核

每个官职有不同考核，不应统一为“政绩分”。

知县考核：

- 钱粮完成。
- 讼案积压。
- 治安稳定。
- 民变风险。
- 上司评价。

御史考核：

- 弹劾命中。
- 清议支持。
- 皇帝容忍。
- 政敌数量。
- 证据可信度。

内阁考核：

- 政策推进。
- 皇帝信任。
- 派系平衡。
- 财政压力。
- 身后风险。

## 11. 派系与人脉

### 11.1 派系不是阵营

游戏中的派系不是永久阵营，而是围绕议题、人物和利益形成的临时联盟。

派系来源：

- 同年：同榜进士互相提携。
- 师门：座师与门生网络。
- 乡党：籍贯和地方利益。
- 清流：以士林名节为核心。
- 实政派：以财政、边防、改革为核心。
- 内廷线：接近皇帝私人信息。
- 地方士绅：控制土地、钱粮和舆论。
- 军功集团：边防与武臣网络。

同一个人物可能同时属于多个网络。玩家可以借其中一张网办事，也可能被另一张网牵连。

### 11.2 人脉牌

重要关系以人脉牌呈现，不一定进入抽牌堆。

例子：

- 座师：每次升迁检定可提供举荐，但会要求派系站队。
- 同年好友：可消除一次弹劾风险，但会留下人情债。
- 精明幕友：政务牌效果 +1，但每年消耗银两。
- 清流领袖：清议牌效果翻倍，但禁止使用明显污点牌。
- 内侍线人：提前看到朝廷事件，但增加清名损失。
- 地方大族：钱粮牌效果提升，但改革阻力上升。

## 12. 政策与长期项目

### 12.1 政策牌

政策牌进入政策区，占用槽位并持续影响局势。

当前 Demo 中，部分卡牌已经带有“政策”标签，例如“严行考成”“裁撤冗费”，但独立政策区、政策槽位和跨季持续结算尚未实装。这一节保留为下一阶段设计目标。

政策牌例子：

| 政策 | 正面效果 | 负面压力 |
|---|---|---|
| 清丈田亩 | 财政健康上升，隐田减少 | 士绅怨恨、短期民怨 |
| 整顿吏治 | 胥吏欺瞒下降 | 行政效率短期下降 |
| 兴修水利 | 灾害损失下降，民心上升 | 财政消耗、贪墨风险 |
| 严行考成 | 政策推进速度提升 | 官员怨恨、数据造假 |
| 开海通商 | 财政收入提升 | 清议争议、走私风险 |
| 裁撤冗费 | 财政压力下降 | 既得利益反扑 |

### 12.2 改革项目结构

大型改革不是一张卡打完，而是一组连续事件。

```text
提出方案
  -> 争取上意
  -> 部院审议
  -> 地方试行
  -> 反弹与谣言
  -> 修正或强推
  -> 阶段成果
  -> 身后评价
```

改革成功不一定让玩家安全。很多改革会制造“生前有用、身后被骂”或“当时失败、后世平反”的结局。

## 13. 卡组流派

### 13.1 清流直臣

核心幻想：以名节和直言对抗权势。

强项：

- 清名、士林评价、弹劾合法性高。
- 污点少，身后评价稳定。
- 能在皇帝信任较高时快速推动清流事件。

弱点：

- 缺钱、缺执行力。
- 容易被孤立。
- 对皇帝猜忌和派系围攻很脆弱。

代表卡：

- 上疏直谏。
- 士林联署。
- 拒收陋规。
- 当廷争辩。
- 辞官明志。

### 13.2 经世能吏

核心幻想：不求漂亮名声，专注把事办成。

强项：

- 治理、财政、工程、考成能力强。
- 能处理灾荒、亏空、民变等硬问题。
- 在地方和部院都有稳定表现。

弱点：

- 容易得罪士绅和同僚。
- 政策压力高，晚年容易被清算。
- 为完成目标常需要灰色妥协。

代表卡：

- 清丈田亩。
- 严行考成。
- 开仓平粜。
- 核查黄册。
- 限期追缴。

### 13.3 圆滑中枢

核心幻想：在皇帝、派系、部院和士林之间维持平衡。

强项：

- 关系修复强。
- 风险转移强。
- 更容易在高位存活。

弱点：

- 改革推进慢。
- 清名与能名都不容易达到极高。
- 一旦失去皇帝信任，盟友可能迅速切割。

代表卡：

- 委婉陈情。
- 两边转圜。
- 延议再定。
- 分功于众。
- 暗中补偿。

### 13.4 权谋枭臣

核心幻想：掌握信息、把柄和任免权，以权力塑造局势。

强项：

- 操作空间大。
- 能快速处理政敌。
- 在党争和中枢局势中强势。

弱点：

- 污点、恶名、恐惧增长快。
- 皇帝猜忌是最大天敌。
- 身后清算风险极高。

代表卡：

- 密折告发。
- 扣留案卷。
- 安插门生。
- 离间朋党。
- 借刀弹劾。

### 13.5 仁政地方官

核心幻想：守一方百姓，宁可升迁慢，也要留下地方善政。

强项：

- 民心、地方稳定、灾害处理强。
- 身后地方评价高。
- 更容易获得祠祀、碑记、百姓请留。

弱点：

- 财政和考成压力大。
- 不擅长中枢党争。
- 容易被上级视为“不识大体”。

代表卡：

- 缓征钱粮。
- 开仓赈济。
- 平反冤狱。
- 约束胥吏。
- 百姓请命。

## 14. 失败与结局

### 14.1 失败不是立刻结束

很多失败会改变人生路线。

| 失败 | 后续可能 |
|---|---|
| 科举失利 | 做幕友、讲学、投靠名臣、继续苦读 |
| 被贬地方 | 积累治理经验，等待复起 |
| 弹劾失败 | 失去清流支持，转向权谋或退隐 |
| 改革失败 | 留下半成制度，进入身后争议 |
| 下狱 | 认罪求生、牵连他人、保全名节 |
| 罢官 | 著书、讲学、经营家族、等待新局 |
| 身体崩溃 | 提前致仕、托付门生、完成遗疏 |

### 14.2 结局评价

结局不是单一分数，而是一段史传式评价。

评价维度：

- 官位：最高做到哪里。
- 政绩：真正改变了什么。
- 名节：是否守住底线。
- 权势：是否影响朝局。
- 民生：百姓是否受益。
- 牵连：是否害了家族、门生、同僚。
- 身后：被追赠、被清算、被禁毁、被平反、被祭祀。

示例结局标题：

- “清节自持，终老林泉”
- “能臣干吏，毁誉参半”
- “权倾一时，身后籍没”
- “救荒有功，十年不迁”
- “党祸牵连，青史留名”
- “生前沉沦，身后追复”

## 15. 当前 MVP 实现范围

### 15.1 MVP 目标

首个版本只验证一个问题：

> 玩家能否通过卡组变化，感受到自己正在形成一种独特的为官人格，并在事件后果中看到这种人格的代价。

### 15.2 已实现切片

当前切片范围：

- 主角：虚构进士一名，避免早期被真实史实绑定。
- 时间：入仕后 12 年，春夏秋冬四季循环。
- 官职：翰林、知县、御史三个阶段，按官评自动升迁。
- 核心事件：18 个官职基础事件，每阶段 6 个。
- 关系事件：8 个，由皇帝猜忌、士林拥戴/反噬、士绅怨恨、胥吏怨恨、师门亏欠、政敌怨恨等状态触发。
- NPC 事件：12 个，由具名人物的羁绊、信任、亏欠、怨恨触发。
- 卡牌：86 张，其中 17 张为初始牌库，知县阶段追加 22 张，御史阶段追加 21 张，其余来自自动成长、NPC 奖励、污点和心病。
- 结局：13 个，含一个强制失败结局。

三个阶段玩法差异：

| 阶段 | 核心体验 | 已实现事件主题 | 追加牌池 |
|---|---|---|---|
| 翰林 | 建立文名、清议与近君关系 | 诏书争议、经筵讲读、馆选派系、诗文招谤、座师请托、同年被参 | 初始牌库，偏奏疏、清议、人情、才学 |
| 知县 | 处理真实地方治理压力 | 县仓亏空、夏汛决口、夺田讼案、钱粮催征、盗匪、胥吏怠工 | 政务、财政、法度、仁政、地方人情、地方污点 |
| 御史 | 进入朝廷斗争和清算风险 | 参劾权臣、边饷冒领、内廷传旨、科场舞弊、税制改革、清算风声 | 弹劾、证据、权谋、清流、近君、朝廷污点 |

### 15.3 已实现系统

- 人物状态：年龄、官职、官阶、官评、属性、特质、资源、名声、污点、主导流派。
- 朝局状态：皇帝信任、士林评价、民心、财政健康、朋党烈度、朝局压力。
- 关系状态：8 个关系节点，包含信任、猜忌、亲近、亏欠、怨恨、畏惧等维度。
- 具名 NPC：12 名 NPC，拥有传记、议程、触发条件、个人状态和事件记录。
- 卡牌循环：抽牌、出牌、弃牌、洗回弃牌堆、保留 1 张手牌、弃 2 张手牌换资源。
- 分支出牌：部分卡牌可选择不同用法，改变目标、费用、风险和后续连招。
- 事件结算：关键阻力压到 4 或以下视为安全，未解阻力转为后患。
- 威胁与反制：强硬或高风险出牌会增加威胁，事件可触发反扑。
- 自动成长：成功或部分成功后自动沉淀新牌、升级现有牌或加入流派奖励。
- 压力与负面牌：压力过高时可能加入心病牌；污点行动会把污点牌加入牌库和污点区。
- 升迁与降调：官评达到阈值会解锁下一官职，严重关系事件失败可能降调或强制结局。
- 结案总结：每季结束后生成弹窗，展示阻力、安全/后患、数值变化、NPC 账、牌库变化和叙事余波。
- 多结局评价：第 12 年后按名声、污点、关系、朝局和流派使用情况判定结局。
- 生平导出：可导出当前人物的名声、朝局、仕途、污点、NPC 评语、史传摘句和生平记事。

### 15.4 暂缓系统

仍未进入当前 Demo 的系统：

- 完整朝代大地图。
- 大规模家族繁衍。
- 复杂军事系统。
- 全朝廷 AI 沙盒。
- 多代继承。
- 全量真实官制还原。
- 独立政策区和跨季政策项目。
- 卡牌升级/异化的主动选择界面。
- 部院官、巡抚/总督、尚书、内阁阶段。

| `src/ui/render.js` | 主界面、关系图、牌库、生平卷、弹窗渲染 |
| `src/main.js` | 启动、按钮绑定、视图切换、存档/重开 |

### 16.2 卡牌结构

当前卡牌字段如下：

```js
{
  id: "memorial_direct",
  name: "上疏直谏",
  type: "奏疏",
  domain: "才学",
  tags: ["清流", "奏疏", "近君", "翰林"],
  cost: { energy: 1, pressure: 1 },
  effect: {
    tracks: { "上意不明": -3, "清议沸腾": -1 },
    world: { emperorTrust: -1, scholarOpinion: 2 },
    tags: { 清流: 1 }
  },
  desc: "以直言求上意，清名见长，但容易触怒天颜。",
  modes: [
    {
      id: "imperial",
      name: "直陈上意",
      desc: "重点压低上意不明。",
      effect: {
        tracks: { "上意不明": -4 },
        world: { emperorTrust: -1 },
        tags: { 清流: 1 }
      },
      setFlags: { direct_advice: 1 },
      reactionRisk: 1
    }
  ]
}
```

已实现的卡牌效果通道：

- `tracks`：降低或增加当前事件阻力。
- `resources`：改变精力、银两、人情、压力。
- `fame`：改变清名、能名、文名、权名、酷名、贪名。
- `world`：改变皇帝信任、士林评价、民心、财政健康、朋党烈度、朝局压力。
- `relations`：改变关系节点的信任、猜忌、亲近、亏欠、怨恨、畏惧。
- `npcs`：改变具名 NPC 的羁绊、信任、亏欠、怨恨、阶段和状态。
- `addStain`：加入污点牌。
- `addCard` / `upgradeFirstCard`：成长或替换卡牌。
- `setFlags` / `bonusIf`：建立连招前置并触发额外效果。
- `reactionRisk`：增加事件威胁，可能触发反制。

关系事件和 NPC 事件在基础字段外增加：

## 17. 界面信息架构

### 17.1 主界面

当前主界面由六块组成：

- 顶部：品牌、视图切换、存档、重开、年份、季节、年龄、官职、官阶、官评、任期目标。
- 左侧：人物状态，包括肖像、籍贯、性格、官职标签、主导流派、属性、资源、名声、污点。
- 中央：当前事务，展示事件描述、参与者、叙事钩子、威胁条、阻力条、当前预估和已用手段。
- 右侧：关系与朝局，展示世界变量、关键关系和风险提示。
- 下方：行动手牌，支持抽牌、出牌、选择分支用法、保留、弃换资源、结束本季。
- 底部：生平记事日志。

### 17.3 结案总结弹窗

每季结束后不再只给一句成败提示，而是生成结案总结：

- 当前结局等级和结案文案。
- 关键阻力是否已压到安全线。
- 未解关键和后患列表。
- 资源、名声、朝局、阵营关系、仕途的数值变化。
- NPC 人情账变化和短叙事。
- 牌库变化、污点和心病变化。
- 自动成长记录。

玩家必须读完总结并点击继续后，才进入下一季。

## 18. 设计风险与约束

### 18.1 避免变成纯数值刷升官

升官必须带来更复杂的责任，而不是单纯奖励。越高位，手里能用的牌更强，但风险、敌人和历史评价也更重。

### 18.2 避免腐败路线成为最优解

污点牌要短期强，但长期污染牌库、增加清算风险，并影响身后评价。腐败可以救急，可以形成权谋流派，但不能无成本滚雪球。

### 18.3 避免清流路线只有道德正确

清流路线要有现实困难：缺资源、推进慢、容易孤立、可能误事。这样道德选择才有重量。

### 18.4 避免历史还原压倒玩法

官制细节服务于玩法。可以使用虚构朝代或半架空剧本，将真实制度转译为清晰机制，而不是让玩家背诵制度名词。

### 18.5 避免卡牌与叙事脱节

每张卡都应有明确叙事含义。玩家打出的不是“造成 3 点伤害”，而是“查封账册”“拜访座师”“上报实情”“私下补账”。数值服务于行为想象。

## 19. 核心卖点总结

- CK3 式人物驱动：玩家经营的是一个人的命运、关系和评价。
- 中国古代官场题材：科举、清议、君臣、党争、地方治理和身后名天然适配策略叙事。
- 卡牌构筑表达人格：卡组不是外部技能树，而是玩家为官方式的沉淀。
- 官职即玩法包：每次升迁不是变强，而是进入新的权力结构。
- 失败生成故事：贬谪、下狱、罢官、清算和退隐都能成为人生转向。
- 名与实的长期拉扯：清名、能名、权势、民心、皇帝信任和污点彼此冲突。

最终目标是让玩家在一局结束时觉得：

> 这不是我把一个角色练满了，而是我亲手把一个人活成了这样。
